# COLOR SYSTEM

## Palette

- Background base: deep navy/slate (`#020617`, `#0f172a`)
- Primary accent: electric cyan (`#22d3ee`)
- Surface cards: translucent slate layers with low-alpha borders

## Semantic Colors

- Healthy: emerald
- Warning: amber
- Critical: red
- Active/informational: cyan/blue

## Constraints

- Avoid rainbow palettes and saturated decorative gradients
- Preserve WCAG-safe text contrast on translucent surfaces
